# Mathematics for Machine Learning — Study Plan

> **Book:** *Mathematics for Machine Learning* by Deisenroth, Faisal & Ong (417 pages)
> **Goal:** Build the mathematical foundation to deeply understand modern ML algorithms.
> **Pace:** ~1 chapter per week, ~45–60 min/day, 5 days/week.
> **Total duration:** ~12 weeks (3 months)

---

## Big Picture: What Is This Book About?

Think of machine learning like baking a cake. Most people just follow a recipe (use scikit-learn, TensorFlow…). This book teaches you **why each ingredient works** — the flour (linear algebra), the heat (optimization), the rising agent (probability). Once you understand that, you can invent your own recipes.

The book has **2 parts**:

| Part | Chapters | Theme |
|------|----------|-------|
| **Part I** | 1–7 | Math tools (the ingredients) |
| **Part II** | 8–12 | ML methods built from those tools (the recipes) |

## How to Use Each Chapter Guide

Each guide file in this folder has:
1. **The Big Idea** — one simple analogy a child could understand
2. **Key Concepts** — explained simply, no jargon dumps
3. **Why It Matters for ML** — direct connection to real algorithms
4. **Daily breakdown** — what to read each day
5. **Self-check questions** — to know if you truly understood

---

## Daily Study Rhythm (45–60 min)

```
10 min  — Review yesterday's notes / morning email tip
20 min  — Read the assigned pages slowly (pencil in hand)
15 min  — Re-derive 1–2 key formulas on paper without looking
10 min  — Write a 3-sentence summary in your own words
```

> **Golden rule:** If you cannot explain it simply, you don't understand it yet. Try to explain every concept as if talking to a 10-year-old.

---

## Prerequisites Check

Before starting, make sure you're comfortable with:
- [ ] Fractions, percentages, basic algebra (solve for x)
- [ ] What a function is: f(x) = 2x + 1
- [ ] Basic geometry: area, angles, Pythagorean theorem
- [ ] What a derivative means (slope of a curve) — Ch 5 will deepen this

If any of these feel shaky, spend a few days on Khan Academy first.
